# NonGrata

Professional registry plugin for Paper Minecraft servers.

## Features

- Interactive GUI
- SQLite
- Future MySQL support
- Player dossiers
- History
- Comments
- Search
- Pagination
- LuckPerms support
- Adventure API
- Modern architecture

---

## Requirements

- Java 21
- Paper 1.21.11+

---

## Build

```bash
mvn clean package
```

Compiled plugin:

```
target/NonGrata-1.0.0-SNAPSHOT.jar
```
